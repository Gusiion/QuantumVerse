Escolha seu banco de dados favorito

Bancos de dados são cidadãos de primeira classe no QuantumVerse. Atualmente oferece suporte a MySQL, PostgreSQL, MongoDB e Redis. Você pode inspecionar os logs do banco de dados ou conectar-se à CLI do banco de dados por meio da interface da web.

In-Browser Client

When you need to quickly run some commands in your database, you can use the web console.

Private or Public Access

By default, databases can be accessed from inside the server, but you can make them public if you want.

Automated Backups

Prevent data loss using periodic backups that are stored on local or remote, S3 compliant, disks.
